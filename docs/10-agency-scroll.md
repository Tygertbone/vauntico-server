## üß™ Agency CLI Invocation

### Command Reference for Agency Operations

#### Onboard a New Client
```bash
vauntico agency onboard \
  --client "brandalchemy.co.za" \
  --modules "audit,distribution,scrolls" \
  --white-label true \
  --branding ./brand-kit.json
```

#### Run Client Audit
```bash
# Comprehensive audit
vauntico audit run \
  --url https://client-site.com \
  --pillars all \
  --brand "Your Agency Name" \
  --output ./reports/client-name-audit.pdf

# Quick SEO + Conversion audit
vauntico audit run \
  --url https://client-site.com \
  --pillars seo,conversion \
  --format web \
  --share-link true
```

#### Generate Workshop for Client
```bash
vauntico workshop create \
  --type masterclass \
  --topic "Email Marketing Mastery" \
  --duration "90min" \
  --client "acme-corp" \
  --brand-kit ./branding/acme.json
```

#### Setup Distribution Pipeline
```bash
vauntico distribution init \
  --client "startup-xyz" \
  --platforms "linkedin,twitter,email,blog" \
  --schedule "mon-wed-fri" \
  --auto-repurpose true
```

#### Generate Client Scroll
```bash
vauntico scroll generate \
  --type case-study \
  --client "success-story-co" \
  --data ./client-data.json \
  --template agency-case-study \
  --output pdf,web
```

#### Bulk Operations
```bash
# Run audits for all clients
vauntico agency audit-all \
  --frequency monthly \
  --notify-clients true

# Generate monthly reports
vauntico agency reports \
  --period "2025-01" \
  --clients all \
  --email-to clients
```

#### Revenue Analytics
```bash
# View agency dashboard
vauntico agency dashboard

# Export revenue report
vauntico agency revenue \
  --period "2025-Q1" \
  --format csv \
  --breakdown by-client
```

## ‚ùì FAQ: Agency Edition

### Can I resell under my own brand?
**Yes.** With Pro or Legacy tier, you have full white-label rights. Your clients never see "Vauntico"‚Äîonly your brand.

### What if a client asks about the technology?
**Positioning options:**
1. "We built our audit engine on proprietary infrastructure."
2. "We partner with enterprise-grade platforms for scalability."
3. "Our system is powered by best-in-class APIs."

All are true. Choose what fits your brand.

### Can I charge more than suggested pricing?
**Absolutely.** Suggested pricing is conservative. If you have premium positioning, charge R20K+ for audits. Your value = your expertise + delivery + results, not just the tool.

### What if I need custom features?
**Legacy tier includes:**
- Quarterly strategy calls with Vauntico team
- Custom feature requests prioritized
- API access for custom integrations
- Dedicated Slack channel

### Can I white-label for other agencies?
**Yes.** If you want to become a Vauntico reseller (offering it to other agencies), contact us for partnership terms. Revenue share available.

### What's the affiliate commission?
**15% recurring** on all referrals. If you refer a Legacy subscriber (R2,999/month), you earn R449.85/month for as long as they stay subscribed.

### Do I need technical skills?
**No.** The dashboard is point-and-click. CLI is optional (but powerful if you have devs). Most agencies use the web interface exclusively.

### Can I offer audits without Legacy tier?
**Yes.** Pro tier includes 1 audit/quarter (2,500 credits total). But for agency-scale volume (10+ audits/month), Legacy is essential.

### What if my client wants direct access?
**Two models:**
1. **Agency-managed:** You run audits, deliver reports (most agencies choose this)
2. **Client-access:** Give them branded portal access, charge monthly seat fee

Both work. Choose based on your service model.

### How do credits work for agencies?
**Legacy tier = 10,000 credits/month** with unlimited rollover. Average audit = 250 credits. So you can run ~40 audits/month, or mix audits + workshops + scrolls.
