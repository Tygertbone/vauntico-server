Overview

The Vauntico Trust Widget is a white-label, embeddable component that displays trust scores with customizable branding and styling. It's designed for Phase 2 B2B API licensing and includes comprehensive KPI tracking.

Configuration Options

{ apiKey: 'your-api-key', // Required: API key for authentication userId: 'user-123', // Required: User ID to fetch score for theme: 'light' | 'dark' | 'auto', // Theme mode (default: 'light') primaryColor: '#007bff', // Primary brand color (default: '#007bff') fontFamily: 'Arial, sans-serif', // Custom font family showLogo: true, // Show Vauntico logo (default: true) showDetails: true, // Show trust factors (default: true) width: '300px', // Widget width (default: '300px') height: 'auto', // Widget height (default: 'auto') borderRadius: '8px', // Border radius (default: '8px') animation: true, // Enable animations (default: true) refreshInterval: 300000, // Auto-refresh interval in ms (default: 5min) onScoreUpdate: (score) => { // Callback when score updates console.log('Score updated:', score); }, onError: (error) => { // Callback for errors console.error('Widget error:', error); } }

API Methods

// Initialize widget const widget = new VaunticoTrustWidget(container, config); // Get current score data const score = widget.getScore(); // Refresh score manually widget.refresh(); // Update configuration widget.updateConfig({ theme: 'dark', primaryColor: '#ff5722' }); // Destroy widget and cleanup widget.destroy();
