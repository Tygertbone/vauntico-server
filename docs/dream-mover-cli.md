## üöÄ Usage Examples

### Basic Generation
```bash
# Generate a blog post
vauntico generate text --prompt "10 tips for productivity" --style blog

# Create an image
vauntico generate image --prompt "cyberpunk city at sunset" --size 1024x1024

# Scaffold a project
vauntico scaffold --template react-vite --name my-app
```

### Batch Processing
```bash
# Process multiple prompts from file
vauntico batch --config generations.yaml --output ./generated

# Pipeline integration
vauntico generate text --prompt "Release notes for v2.0" | git commit -F -
```

### Template Management
```bash
# Save a generation template
vauntico template save --name "weekly-newsletter" --config template.json

# Use saved template
vauntico template use weekly-newsletter --variables data.json
```
