## The Credit Economy: How Power Flows

**Credits are your creative fuel.** Every action in Vauntico consumes credits based on complexity and compute.

### Credit Costs (Examples)

| Action                      | Credits |
| --------------------------- | ------- |
| Generate landing page       | 50      |
| Create workshop module      | 100     |
| Run technical audit         | 250     |
| Generate scroll (long-form) | 75      |
| Deploy full funnel          | 200     |
| Custom component generation | 30-150  |

**Rollover Rules:**

- **Starter:** No rollover (use it or lose it)
- **Pro:** Roll over up to 1,000 credits
- **Legacy:** Unlimited rollover (build your vault)

**Overage Protection:**

- Set credit limits to avoid surprise charges
- Get warnings at 80% and 100% usage
- Pause auto-regeneration if needed
