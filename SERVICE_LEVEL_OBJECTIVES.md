# Vauntico Service Level Objectives (SLOs)

## Availability

- Target: 99.9% uptime per calendar month
- Measurement: Prometheus uptime metrics

## Latency

- Target: < 200ms average response time for core APIs
- Measurement: Grafana dashboards

## Error Rate

- Target: < 1% failed requests
- Measurement: Sentry error tracking

## Incident Response

- Critical issues acknowledged within 15 minutes
- Resolution or rollback within 1 hour
